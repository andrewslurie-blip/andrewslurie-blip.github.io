About this project

The data presented measures the status in 2025 of the land area of forest in a country as a percentage of total land area in a country. This directly connects to the UN’s sustainable goal 15, life on land, which has the goal to “Protect, restore and promote sustainable use of terrestrial ecosystems, sustainably manage forests, combat desertification, and halt and reverse land degradation and halt biodiversity loss.” By having baseline forest data and comparing trends over time, the UN is able to directly track the impacts of deforestation and land degradation. More information about the sustainable development goals and the data associated can be found at the UN Statistics Division website.

This map is a choropleth map. A choropleth map is a type of informational map that relays a certain data set pertaining to countries in a clear, colorful format. Using different colors belonging to ranges of forest coverage, (0-20%, 20-40%) countries are classified in a range and set to a certain shade of a color. The benefit of this is that the map is extremely easy to understand, and makes for a good story, as countries’ differences are often inflated. The downside to this simplicity is, however, that regional differences are not visible. In the case of forests specifically, forests transcend borders, and this can simply not be seen on country based data.

While making this map I learned how to fuse a map file and a data file together, allowing me to create color shades and project the data onto a map. I also learned how to deal with data types, folders, and other computer tricks. Most of all, I learned how to troubleshoot errors, and that sometimes the only way is AI.
